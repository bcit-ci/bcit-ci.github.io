CLI Class¶

Class CLI

Tools to interact with that request since CLI requests are not static like HTTP requests might be.

Portions of this code were initially from the FuelPHP Framework, version 1.7.x, and used here under the MIT license they were originally made available under.

http://fuelphp.com

CodeIgniter\CLI\CLI
init()¶

Static “constructor”.

input([string $prefix])¶
Parameters:
  • $prefix (string) –

Get input from the shell, using readline or the standard STDIN

Named options must be in the following formats: php index.php user -v –v -name=John –name=John

prompt()¶

Asks the user for input. This can have either 1 or 2 arguments.

Usage:

// Waits for any key press CLI::prompt();

// Takes any input $color = CLI::prompt(‘What is your favorite color?’);

// Takes any input, but offers default $color = CLI::prompt(‘What is your favourite color?’, ‘white’);

// Will only accept the options in the array $ready = CLI::prompt(‘Are you ready?’, array(‘y’,’n’));

  • $background (string) –

Outputs a string to the cli. If you send an array it will implode them with a line break.

Parameters:
  • $text (string) –

Returns the given text with the correct color codes for a foreground and optionally a background color.

getWidth([int $default])¶
Parameters:
  • $default (int) –

Attempts to determine the width of the viewable CLI window. This only works on *nix-based systems, so return a sane default for Windows environments.

getHeight([int $default])¶
Parameters:
  • $default (int) –

Attempts to determine the height of the viewable CLI window. This only works on *nix-based systems, so return a sane default for Windows environments.

showProgress([int $thisStep[, int $totalSteps]])¶
Parameters:
  • $thisStep (int) –
  • $totalSteps (int) –

Displays a progress bar on the CLI. You must call it repeatedly to update it. Set $thisStep = false to erase the progress bar.

wrap([string $string[, int $max[, int $pad_left]]])¶
Parameters:
  • $string (string) –
  • $max (int) –
  • $pad_left (int) –

Takes a string and writes it to the command line, wrapping to a maximum width. If no maximum width is specified, will wrap to the window’s max width.

If an int is passed into $pad_left, then all strings after the first will padded with that many spaces to the left. Useful when printing short descriptions that need to start on an existing line.
